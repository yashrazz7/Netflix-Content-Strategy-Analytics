# 🎬 StreamPulse: Global OTT Content & Viewership Insights (Netflix)

### 📊 Project Overview
This repository contains an enterprise-grade data analytics project focused on deciphering the content strategy, distribution architecture, and production trends of Netflix. Using Tableau Desktop and Exploratory Data Analysis (EDA) methodologies on metadata containing over 10K+ records, this project transforms raw corporate metrics into executive-level strategic insights.

---

### 🔍 Key Engineering & Strategic Insights Covered
1. **Content Mix Optimization:** Comprehensive breakdown and statistical distribution of Movies versus TV Shows reflecting shifting user engagement paradigms.
2. **Geographical Production Density:** Global mapping and heatmaps tracking content hubs and the exponential 300% growth of non-English regional content.
3. **Demographic Ratings Architecture:** Granular classification of content maturity ratings (TV-MA, TV-14, PG-13) segmented by targeted viewer demographics.
4. **Temporal Trajectory Analytics:** Deep dive into release timelines post-2015, analyzing historical library expansion and scheduling patterns.
